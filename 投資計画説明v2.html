📝 経営戦略の概要

📅 令和7年4月15日に発表された、佐世保市水道事業の10年間の経営戦略。
目的は、水道供給の持続を通じて市民生活と都市の成長を支えること。
📌 戦略は「投資計画」中心に構成され、Phase1（最初の10年）では、基幹インフラに重点を置く。

📉 背景と課題

🔍 人口減少 ➜ 水需要の減少 ／ 一斉更新時期 ➜ 老朽化施設が多数
🧩 結果：施設の統廃合、ダウンサイジングが必要に。
💡 現在の課題：施設が多く、古い。
🎯 対応策：長寿命化とリスクマネジメントにより、安全性とコスト削減を両立。

🗺️ ロードマップと方向性

🚧 基幹施設を優先整備し、小口径管路はモニタリングで維持管理。
🔁 施設統廃合は長期戦略の中で段階的に実施。
⚙️ 給水サービスの維持が全体の最優先目標。
そのために、戦略的にリスクと投資を分配。

📐 投資計画の基本的な考え方

🏢 他都市より施設が多く老朽化。
🧭 「すべてを更新」は不可能 ➜ だからこそ、負担軽減とコスト最適化が必要。
🧰 具体策：
  • 施設数を減らす
  • 施設規模を小さく
  • ライフサイクルコストを抑制

🛡️ 安全性の確保と整合性

🔍 投資額の最小化は大切だが…
✅ 給水の安全性と都市政策との整合性が大前提。
🧩 リスクの取り扱い方：
  • 都市計画に沿った設備投資
  • 災害・老朽化リスクに備える
  • 統廃合や改修は政策の枠内で実施

🏞️ ダムの老朽化とリスク

⏳ 山の田・転石・菰田・相当・川谷・下の原…
ダムの多くが 築60年以上。最古の山の田ダムは明治40年建設！
⚠️ 決壊時の被害：下流の家屋・命に重大なリスク。
特に取水設備は故障時に 大規模断水を引き起こす。
🔧 山の田ダムでは補強工事済（H18年）。
今後の抜本的更新は 水源確保後に開始予定。

🚰 浄水施設の現状と対応

📉 小規模浄水場（25箇所以上）が老朽化。
統廃合の対象とし、大規模施設に集約予定。
🏢 広田・柚木など主要浄水場の更新時期が近づいており、現地更新＋統合型施設へ。
🛠️ 山の田浄水場はH27年に更新済。
最新の膜ろ過方式＋DBO方式を採用し、今後の中核に。

📦 浄水施設別の方針まとめ

🧱 小規模浄水場 ➜ 廃止・統合による削減を基本。
🧪 大規模浄水場 ➜ 予防保全型で更新を進める。
🔄 北部エリアへは 佐世保地区から送水を可能にし、段階的に小規模施設を廃止へ。

🚛 送水・配水施設の戦略

🧮 管路の総延長：2,323km。うち多くが老朽化。
🛣️ 更新は年15kmペースで実施。
🌀 配水ループ化で、事故発生時のリスク分散と迅速復旧が可能に。
🔁 基幹管路は予防保全型更新、小口径管路はモニタリング＋事後対応型へ。

🔗 関連重点事業

  • 🏗️ 山の田水系統合：老朽2施設の統合再構築
  • 🚧 広田水系管路更新：Φ700〜1000で南部の大動脈を強化
  • 🚰 小佐々地区送水管整備：北部エリアの水源不足とサービス不均衡の是正

📡 リスクマネジメント事業

🧠 モニタリング地点の拡充＋管網のブロック化で、漏水の早期検知・対応力アップ！
💻 水運用総合監理システムで、管路情報・流速・水圧などを統合管理
🔧 初動対応の迅速化を図るため、資器材や技術継承の整備も同時実施

🛠️ 安全性確保：優先事業

🏞️ 洪水吐きの改修（歌ケ浦・山の田）で、豪雨時の堤体破損リスクを軽減
💧 河川への安全放流が可能に。
⛲ 石木ダム完成後、老朽ダムの取水設備を順次更新・改修予定。

🚰 導水管の更新

🔁 菰田導水管・吉福導水管：年次ごとに更新を進行中（延長10,000m超）
💧 ダム→浄水場の命綱、全市給水の基盤！
📅 更新計画は20年スパンで継続的に進行。健全度評価による優先順位でローリング更新。

📈 投資フェーズと目標

🌀 Phase 1：基幹施設優先、更新量15km／年＋統合基盤整備。
🧩 Phase 2：小口径施設や枝葉部分の施設削減・整備。
⏳ 老朽化対策の初期段階、リスクコントロールしつつ準備を進行中。

📉 数量目標（10年後）

📌 浄水場数：28 ➜ 24（−4）
📌 配水池：327 ➜ 297（−30）
📌 ポンプ所：155 ➜ 125（−30）
📌 管路延長：+7km（統合に伴う一時的増加）
💧 漏水量は 7,000㎥／日 を維持、修繕件数は 500 ➜ 640件に増加予測（対応力強化）

💰 投資総額と内訳（10年間）

💸 合計：959億円
  • 水源確保：321億円
  • 安全確保：407億円
  • 更新需要削減：136億円
  • リスクマネジメント：41億円
  • その他：54億円
📌 この費用は、経営方針に基づいた最小限の投資に絞った結果。
❗ 先送りできない案件を中心に編成。
